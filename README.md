# Raj Sharma Portfolio

A personal portfolio website built with React, TypeScript, and Vite.

## Features

- Responsive design
- Social media links
- Contact form
- Portfolio section (under construction)

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
